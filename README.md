# ProgressReport
ProgressReport for patients
1. 建立名為"習慣養成"的資料夾於執行檔的目錄底下
2. 習慣養成裡面的按年份創立資料夾如"習慣養成\11308"、"習慣養成\11309"...
3. 每個年份的資料夾放入對應的excel檔案，命名為"習慣養成統計—叔珍組11308.xlsx"、"習慣養成統計—珮茹組11308.xlsx"...
4. excel檔案裡面的工作表命名須為病友的"2"個字的名字如"家慶"、"宗彥"...
5. 工作表裡的總分需位於"AG19"格，否則程式抓不到
6. 程式會自動跳過"總成績"的工作表
7. 程式執行上若遇到任何問題，請保留執行檔底下的"logfile.txt"這個檔案
